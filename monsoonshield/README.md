# MonsoonShield — Frontend Application

AI-powered monsoon preparedness portal built with Next.js 16, React 19, TypeScript, and Google Gemini AI.

**Live:** [https://monsoonshield-kappa.vercel.app](https://monsoonshield-kappa.vercel.app)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 16 (Turbopack)
- React 19
- TypeScript 5
- Tailwind CSS v4
- Google Gemini 2.0 Flash
- Recharts (charts)
- Lucide React (icons)

## Features

12 screens: Home, Dashboard, AI Chat, SOS, Reports, Preparedness, Shelters, Weather, Family, Command Center, Insurance, Health Center.

## Build

```bash
npm run build
npm start
```
